```
a*Ax + b*Bx = Cx
a*Ay + b*By = Cy
```

```
a*Ax*By + b*Bx*By = Cx*By
-a*Ay*Bx - b*By*Bx = -Cy*Bx
->
a*Ax*By - a*Ay*Bx = Cx*By - Cy*Bx
->
a = (Cx*By - Cy*Bx)/(Ax*By - Ay*Bx)
```

```
a*Ax*Ay + b*Bx*Ay = Cx*Ay
-a*Ay*Ax - b*By*Ax = -Cy*Ax
->
b*Bx*Ay - b*By*Ax = Cx*Ay-Cy*Ax
->
b = (Cx*Ay-Cy*Ax)/(Bx*Ay - By*Ax)
```
